遇到的问题

在网页内如何跳转，：CSS 和 HTML 之间的一个常见问题，就是不同的 CSS 规则之间发生冲突。 当您使用第三方代码时，问题可能大到难以修补。 例如，你可能正用着一个 CSS 框架，并发现它使用的一个类名与你已经用于某处的类名冲突。

如何解决

指的是优先级最高的样式有效，其中行内样式>内嵌式样式|链接外部样式，即行内样式的优先级最高，而内嵌式样式和链接外部样式的优先级由它们出现的位置决定，谁出现在后面谁的优先级就高。

哪些未解决

主要是针对继承样式，越靠近格式化的元素的父类样式，优先级越高的问题没有完全弄懂。

总结

HTML 是用来描述网页的一种语言。 HTML 指的是超文本标记语言 (Hyper Text Markup Language) HTML 不是一种编程语言，而是一种标记语言 (markup language) 标记语言是一套标记标签 (markup tag) HTML 使用标记标签来描述网页HTML 是一种在 Web 上使用的通用标记语言。HTML 允许你格式化文本，添加图片，创建链接、输入表单、框架和表格等等，并可将之存为文本文件，浏览器即可读取和显示。通过这次的实验，我在html上的了解更进一步。
